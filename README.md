# Migueli Guru Finances

Private app for tracking my Crypto, ETF, and Stock investments.

## Technical Technical

- **Effect**: Trying out **Effect**, a powerful TypeScript library designed to help developers easily create complex, synchronous, and asynchronous programs.

- **Vercel BotID**: an invisible CAPTCHA that protects against sophisticated bots without showing visible challenges or requiring user action. Protects page's and api's routes (`instrumentation-client.ts`)

## Code Quality Tools

This project uses automated code quality tools to maintain consistency:

- **Prettier**: Ensures consistent code formatting
- **Vitest**: Unit testing
- **Knip**: Detects unused files, exports, and dependencies
- **TypeScript**: Provides type safety
- **Husky**: Runs pre-commit hooks automatically
- **SonarQube Cloud**: Continuous code analysis for bugs, vulnerabilities, and code smells
